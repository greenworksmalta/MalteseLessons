# MalteseLessons

Mobile-friendly Maltese language study app, designed for learners with dyslexia and ADHD.

Each lesson is a self-contained HTML page with:

- Native-quality Maltese audio on every word and sentence (Azure Neural TTS — `mt-MT-GraceNeural` / `mt-MT-JosephNeural`)
- Duolingo-style interactive exercises (tap-the-translation, listen-and-choose, tap-to-build, match pairs, fill-the-blank multiple-choice)
- Sectioned by category — Phrases, Alphabet, Grammar, Days, etc. — so you can drill what you need
- Dyslexia-friendly typography, generous spacing, one thing per screen, instant feedback, progress dots
- Works on phone via "Add to Home Screen", offline after first load

Hosted via GitHub Pages.
